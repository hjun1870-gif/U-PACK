창고 재고조사표 시각화 시스템

B2, B1, B3 등 창고 도면을 격자로 시각화하고 마우스 클릭으로 재고를 관리합니다.

오프라인 모드
적재 품목 수: 0
총 파렛트(P) 수량: 0
총 박스(B) 수량: 0
도면 규격: -

재고조사표 엑셀 파일을 업로드해 주세요

상단의 '엑셀 파일 업로드' 버튼을 누르거나 파일을 드래그 앤 드롭하면 창고 도면이 즉시 렌더링됩니다.

📱 스마트폰 / 태블릿으로 접속 및 수정하는 방법

  1. PC와 모바일 기기를 동일한 Wi-Fi 공유기에 연결합니다.
  2. PC의 명령 프롬프트(cmd) 등에서 ipconfig를 입력하여 PC의 로컬 IP 주소(예: 192.168.x.x)를 파악합니다.
  3. 스마트폰 브라우저 주소창에 http://[확인한 PC 로컬 IP 주소]:8010/index.html를 입력해 접속합니다.
  4. 손가락으로 도면을 드래그하며 랙을 터치하면, 모바일 터치 패널로 간편하게 재고를 조회하고 수정할 수 있습니다.

⚙️ Supabase 실시간 동기화 설정

Supabase 프로젝트 정보를 기입하면 PC와 모바일 간 실시간으로 데이터가 연동됩니다. 입력하지 않으면 로컬 엑셀 파일 업로드 모드로 작동합니다.
