재고조사표 엑셀 파일을 업로드해 주세요

상단의 '엑셀 파일 업로드' 버튼을 누르거나 파일을 드래그 앤 드롭하면 창고 도면이 즉시 렌더링됩니다.

📱 스마트폰에서 조회·수정하기

방법 1 (권장) — 인터넷만 있으면 어디서나 접속

  1. 스마트폰 브라우저에서 https://hjun1870-gif.github.io/U-PACK/ 를 엽니다.
  2. 상단 ⚙️ DB 설정에서 PC와 동일한 Supabase URL·Anon Key를 입력합니다.
  3. PC에서 엑셀을 업로드하면 DB에 자동 동기화됩니다. 모바일에서 바로 확인·수정할 수 있습니다.

방법 2 — 같은 Wi-Fi (로컬 PC 서버)

  1. PC와 모바일을 동일 Wi-Fi에 연결합니다.
  2. PC에서 ipconfig로 로컬 IP(예: 192.168.x.x)를 확인합니다.
  3. 모바일에서 http://[PC IP]:8010/index.html 로 접속합니다.

재고 상세 수정

⚙️ Supabase 실시간 동기화 설정

Supabase 프로젝트 정보를 기입하면 PC와 모바일 간 실시간으로 데이터가 연동됩니다. 입력하지 않으면 로컬 엑셀 파일 업로드 모드로 작동합니다.

켜두면 업로드한 엑셀 데이터가 Supabase에 자동 반영되어, 모바일에서도 바로 확인·수정할 수 있습니다.
